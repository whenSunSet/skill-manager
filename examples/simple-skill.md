# Greeting Skill

This skill helps create appropriate greetings for different contexts and audiences.

## Overview

Greetings vary based on:
- Formality level (formal, casual, intimate)
- Time of day
- Relationship
- Cultural context

## Common Greeting Patterns

### Formal Greetings

Use in professional contexts:
- "Good morning/afternoon/evening, [Name]"
- "Dear [Name],"
- "Hello [Name],"

### Casual Greetings

Use with friends and peers:
- "Hi [Name],"
- "Hey [Name],"
- "What's up?"

### Time-Based Greetings

- Morning (6am-12pm): "Good morning"
- Afternoon (12pm-6pm): "Good afternoon"
- Evening (6pm-10pm): "Good evening"
- Night (10pm-6am): "Good night" (farewell)

## Quick Examples

**Email:**
```
Dear [Name],

I hope this email finds you well.
```

**Meeting:**
```
Good morning everyone. Let's get started.
```

**Chat:**
```
Hey! How's it going?
```
